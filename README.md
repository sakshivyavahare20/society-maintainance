# Society Maintenance Tracker

## Overview

Society Maintenance Tracker is a simple frontend prototype for managing apartment society maintenance complaints. It provides separate interfaces for Residents and Admins to demonstrate the basic workflow of complaint management.

---

## Features

### Resident
- Login page
- Raise a maintenance complaint
- Select complaint category
- Add complaint description
- Upload an optional photo
- View complaint list
- View complaint status history
- View society notices

### Admin
- Dashboard showing complaint statistics
- View all complaints
- Update complaint priority
- Update complaint status
- Add admin notes
- Post society notices
- View notice board

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in any web browser.

No installation or server setup is required.
